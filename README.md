# Тестовое задание: Блок апелляции IELTS

## Описание
Адаптивная верстка блока для подачи апелляции на результаты экзамена IELTS.

## Функционал
- Скачивание PDF-форм
- Загрузка отсканированных заявлений  
- Адаптивный дизайн (от 360px)
- Интерактивные кнопки загрузки файлов

## Технологии
- HTML5
- CSS3
- JavaScript (Vanilla)

## Запуск
Открыть `index.html` в браузере
